# Job Tracking System
A web application developed using Spring Boot and Thymeleaf to manage and track job applications.
## Features
- Add, update, and delete job details
- View list of applied jobs
- Track job status
## Tech Stack
- Java
- Spring Boot
- Thymeleaf
- MySQL
## Description
This project helps users organize and manage their job applications efficiently. It follows MVC architecture and provides a simple user interface for interaction.
